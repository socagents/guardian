| **Malware verdict** | 8 | (Palo Alto WildFire + Local Analysis) | venus.exe sandcat binary + Akagi64.exe |
| **Suspicious Process Creation** | 10 | (behavioral) | venus.exe parent process |

**Per-ability alert volume ranking** (from highest Cortex-detected impact):

1. **PowerShell-encoded ability execution** — 39 alerts. Every Caldera Windows ability uses `powershell.exe -EncodedCommand`, generating Script Engine Activity / Powershell Activity / Credential Gathering Protection alerts.
2. **Sandcat venus.exe beacon** — 20 alerts. The Caldera implant itself generates WildFire Malware + Local Analysis Malware + Suspicious Process Creation + BITS Jobs alerts whenever it runs.
3. **Akagi64 Fodhelper UAC bypass** — 2 alerts. Palo Alto WildFire flags Akagi64.exe binary as known malicious.
4. **PowerKatz Mimikatz args** — 2 alerts. Cortex's "PowerShell runs with known Mimikatz arguments" + "Credential Gathering Protection" both fire.

Plus 13 YARA matches (incidents 1796-1808) for "Ransomware_Note_And_Extensions" — generated by Compress-Archive of the decoy files. **The decoy fix from v0.5.83 actually triggered Cortex's YARA ransomware rule** — the synthetic PHANTOM-LAB-DECOY files + their zip patterns match Cortex's known ransomware-stage YARA signatures. Useful detection density.
