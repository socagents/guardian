# Review SOC Coverage

Use this skill when the operator asks for a SOC coverage summary,
executive report, analyst report, or gap analysis.

Procedure:

1. Call `phantom_generate_coverage_report`.
2. Summarize ATT&CK coverage by tactic and technique.
3. Highlight missed detections, noisy fields, and log source gaps.
4. Recommend rules, enrichment, parser changes, or new scenarios.
5. Include links or IDs for simulations and validation runs when
   available.
