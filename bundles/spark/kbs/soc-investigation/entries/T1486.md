# Data Encrypted for Impact (Ransomware)

**What it is:** Adversary encrypts files across hosts to disrupt operations and extort payment — the terminal stage of most ransomware intrusions.

**How it manifests:**
- Pre-encryption tells (catch it HERE): `vssadmin delete shadows /all`, `wmic shadowcopy delete`, `wbadmin delete catalog`, `bcdedit /set recoveryenabled no` (inhibit recovery, T1490); stopping backup/DB/AV services (`net stop`, `taskkill`).
- Encryption burst: a single process performing mass file reads+writes+renames across directories and network shares at high rate; new uniform file extension (`.lockbit`, `.encrypted`, random) appended fleetwide.
- File artifacts: ransom notes (`README.txt`/`HOW_TO_DECRYPT.html`) dropped in every directory; the encryptor binary in `%TEMP%`/`%PUBLIC%`/a share.
- Often deployed at scale via PsExec/GPO/scheduled task across many hosts near-simultaneously.

**Investigation steps (in order):**
1. Identify the encryptor process + binary hash on a hit host; capture command line and parent.
2. Look for the shadow-copy deletion / backup-tamper events immediately preceding encryption — that timestamp anchors the attack.
3. Determine the deployment mechanism (PsExec service install 7045, GPO, scheduled task) to find ALL targeted hosts before they finish.
4. Trace backward to dwell-time activity: initial access, credential dumping (T1003), lateral movement (T1021), and likely prior data exfil (double-extortion — check egress).
5. Isolate affected + at-risk hosts; preserve a sample of encrypted files + the ransom note for IR/decryptor matching.
6. Identify the ransomware family from the note/extension/IOCs to scope known TTPs.

**Data sources:** EDR process + file-modification events (mass-rename detection), Windows Security/System (7045 service install, service-stop), 4688 for `vssadmin`/`wbadmin`/`bcdedit`, file-server audit logs, backup-system logs, egress/NetFlow (for prior exfil).

**Pivot/related:** T1490 (inhibit system recovery), T1489 (service stop), T1021 (deployment lateral movement), T1003.001 (creds for spread), T1567/T1041 (double-extortion exfil).
